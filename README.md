# E-commerce Website

Dự án **E-commerce Website** được xây dựng nhằm mô phỏng một hệ thống bán hàng trực tuyến đầy đủ tính năng.  
Bao gồm 2 giao diện chính: **Admin Dashboard** và **Client/User Website**.

---

## Công nghệ sử dụng

### **Frontend**
- **Next.js 14**
- **TailwindCSS**
- **TypeScript**


### **Backend**
- **Node.js**
- **Express.js**
- **RESTful API**
- **JWT Authentication**

### **Database**
- **MongoDB + Mongoose**

### **Storage (Media Upload)**
- MongoDB GridFS hoặc Cloud Storage (Cloudinary / Supabase / AWS S3)

---

## Tính năng chính

### **Người dùng**
- Đăng ký / Đăng nhập / Xác thực
- Xem danh sách sản phẩm
- Tìm kiếm, lọc sản phẩm
- Thêm vào giỏ hàng
- Đặt hàng, thanh toán
- Xem trạng thái đơn hàng

### **Admin**
- Quản lý sản phẩm (CRUD)
- Quản lý danh mục
- Quản lý người dùng
- Quản lý đơn hàng
- Dashboard thống kê
